# python_roguelike
A rogue-like game and engine, written completely in python. 

# Direction
This project was created to explore the game development capability of Python outside of Visual-Novels or If-statement extravaganzas. 

#How to Install & Play
- In your preferred IDE, import the project files to your environment.

- Either install the required packages manually, or run:
    - `pip install -r requirements.txt` (be sure to cd into the project folder if the project root isn't accessible)
    
    - for Linux; add `user` if you are getting a permission error. You don't have to use sudo.
    
    - If you have multiple python versions installed on your machine, make sure to:
        - The project root is located in a project folder with Python 3.11+ set to the venv interpreter
        -If you're having issues, try `pip3` instead. You could also use the specific path: `python pip install....`

- Run main.py
